Lista Enlazada Dinámica con Ciclo While

Este repositorio contiene la implementación de una Lista Enlazada Simple desarrollada en C++. El objetivo del programa es demostrar el uso de punteros, estructuras (`struct`) y gestión de memoria dinámica mediante el operador `new`.

Descripción del Proyecto

El programa permite al usuario ingresar una serie de números enteros de forma indefinida. Cada número se almacena en un nodo creado en tiempo de ejecución, conectándose al final de la lista. El proceso continúa hasta que el usuario decide detenerse.

Características principales:
- Memoria Dinámica: uso de `new` para reservar espacio solo cuando es necesario.
- Inserción al Final: mantiene el orden de ingreso de los datos.
- Cálculos Estadísticos: conteo total de nodos creados.
    * Suma acumulada de los valores.
    * Cálculo del promedio aritmético.
